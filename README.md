# DISCUSSION APP

**INTRODUCTION**
~~~
Discussion App is an online-meeting web app having live-video call, text-
messaging, white-board using JavaScript, Node.js, Mongodb,
Bootstrap, ejs
~~~

### Pre-Requisites

~~~
You Must have Node.js, mongoDB installed in your system
~~~

### How to Use
**Step: 1**
~~~
clone this repo
~~~

**Step: 2**
~~~
sudo npm install
~~~

**Step: 3**
~~~
node app.js
~~~
